# Flask App Demo | ITE 367
#### Description: 
This is a demo Flask app for CS/ITE 367 to help in learning about DevOps principles. This project involved creating virtual machines (VMs), running CentOS, to deploy a web app using Docker. 
#### Skills Learned: 
During this project I got to work extensively with Linux VMs inside of VMWare Workstation including, creating snapshots, restoring snapshots, and networking. Once the machines were properly configured I was able to being using and learning about the Flask web framework, some basic Python, and Docker containers.
