# Flask Web Application Deployment | ITE 367

## Project Overview

This project demonstrates the deployment of a basic Flask web application within a virtualized environment. The objective was to apply foundational DevOps concepts by provisioning virtual machines, configuring a Linux server environment, containerizing the application with Docker, and successfully deploying it.

The application was developed using the Flask framework and deployed inside a Docker container running on a CentOS-based virtual machine.

---

## Technologies Used

- **VMware Workstation** – Virtual machine management  
- **CentOS Linux** – Server operating system  
- **Docker** – Containerization platform  
- **Flask** – Python web framework  
- **Python** – Backend development  

---

## Skills & Concepts Demonstrated

### Virtualization & System Administration
- Created and configured Linux virtual machines in VMware Workstation  
- Managed VM snapshots (creation and restoration)  
- Configured VM networking  
- Performed Linux system configuration and command-line administration  

### Application Development
- Developed a basic web application using Flask  
- Applied foundational Python programming concepts  
- Structured a simple web application backend  

### Containerization & Deployment
- Wrote a Dockerfile to containerize the application  
- Built Docker images  
- Deployed and managed Docker containers  
- Verified application availability through browser access
